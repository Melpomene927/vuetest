# vuetest

Using Vue.js + uni-app for testing

* npm install -g @vue/cli@4.5.15
* vue create -p dcloudio/uni-preset-vue **<專案名稱>**
* cd **<專案名稱>**
* npm i @dcloudio/uni-helper-json
* vue add @vue/cli-plugin-eslint
* npm run serve `啟動網站`
